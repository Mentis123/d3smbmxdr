How to use this kit Read first

One-pass NBP generation will NOT produce a coherent 745×4474 infographic. Build it in tiles, then stitch.

  1. Start with the style bible. Copy the bible prompt, run it once in Nano Banana Pro at 9:16. Pick the variant with the strongest gradient + isometric glass feel. This becomes your style anchor for every subsequent panel.
  2. Upload the style anchor + the matching reference chop (refs/0X_*.jpg) as image inputs when running each panel prompt. NBP's multi-image conditioning is how you hold consistency across all 8 tiles.
  3. Generate panels 1 through 8 in order. For each: use the panel's prompt, attach (a) your Pass-1 style anchor and (b) the corresponding ref chop. Generate 4 variants per panel; keep the one that best matches the anchor.
  4. NBP renders the titles, you overlay the body copy. Each prompt asks NBP for real headlines, sub-heads and short labels — things it renders well. Body paragraphs are NOT rendered by NBP at all; instead each prompt reserves clean, empty negative space on the background so you can overlay Figma type onto a consistent, seamless gradient.
  5. Stitch in Figma. Drop panels into a 745-wide artboard, overlap 40–60px per seam, blend with gradient masks so background transitions are seamless. See the stitch table for target heights.
  6. Final pass: type overlay → export → QA on mobile → upload to the AI Practice SharePoint + dashboard.
Content grounding: The copy is pulled from the d3brain vault — Mark Pattie's Envision → Transform → Operate framework, the AI Squads MVS model, the AAF five-layer stack, and the five-service catalogue (Lean Sprint, Multi Agent Strategy, Factory Pod, Governance, Scalable Foundation). Tier 3 renamed from "Delivery Pod" → "Factory Pod" per Alenka V3.3 (2026-04-15 early-AM). Squads are how we deliver; the capability is what you buy. If Mark, Suzanne, or Scott redline these, update panels 3–7 and re-run.
Guardrails: Do NOT reference Brisbane Catholic Education. Do NOT use projection numbers as realised numbers (see feedback_poc_vs_case_study.md). The "150+ Copilot Projects" and "#1 ANZ MS Partner of the Year 2025" creds are the only hard numbers approved for external use.
